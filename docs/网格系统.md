## git分支命令
- 本地新建button-and-input分支

    ```git branch button-and-input```
-  ```git add .```
- ```git commit -m 'master提交'```
-  本地分支同步推送到远程分支
    - 把本地的button-and-input分支推到远程的 
    button-and-input分支

        ```git push origin button-and-input:button-and-input```
